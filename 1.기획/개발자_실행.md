# EduChain AI — 개발자 실행 (로컬)

> **앱 루트:** 저장소의 `EduChain_AI/` 폴더.  
> **기획 문서:** 상위의 `1.기획/` 폴더.

---

## 최초 세팅 (한 번)

PowerShell 예시:

```powershell
cd c:\project\KIT_Contest\EduChain_AI
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy .streamlit\secrets.toml.example .streamlit\secrets.toml
```

`secrets.toml`에 `GEMINI_API_KEY`, Firebase 관련 키 등을 채웁니다. 자세한 항목은 **`1.기획/연동필요.md`** 를 본다.

---

## 매번 실행

```powershell
cd c:\project\KIT_Contest\EduChain_AI
.\.venv\Scripts\Activate.ps1
streamlit run Home.py
```

브라우저에서 앱이 열리면 멀티페이지(로그인·관리·교사·학생 등)로 이동해 시연한다.
